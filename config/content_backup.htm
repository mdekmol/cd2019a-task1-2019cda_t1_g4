SSH步驟

1.利用cmd.exe(小黑窗)

輸入指令:git config --edit --system

叫出vi 編輯器然後只能利用上下左右控制

按a進入編輯後左下角會顯示-- INSERT --

2.進入後會如下圖所示

按BACKSPACE

把紅框中的 helper=manger 給刪除

接下來輸入 冒號( : )然後輸入WQ(儲存後並關閉)

PS : 如不儲存的話直接輸入EXIT(直接離開)

3.金鑰製作

到cmd.exe(小黑窗)輸入 sh_yen就會進入git 的shell

輸入     ssh-keygen -t rsa -b 4096 -C "This is my key"

PS:    this is my key 請輸入你要的名稱例如：學號

然後會出現一段英文會叫你按ENTER鍵

然後就按下ENTER鍵

金鑰內容會出現在   y:\home\.ssh\id_rsa與id_rsa.pub

然而我們需要的是 id_rsa.pub裡的內容

PS:上圖中2個檔案中一個很長一個較短選短的那個

利用Scite打開檔案並複製裡面之內容複製至github

進入後

把複製的字串給貼入後儲存至你們小組倉儲切換至SSH再進行clone一次

這樣以後就不用輸入帳號密碼來進行推送

在公共場所也相對安全

PS : SSH是認髓身碟的倉儲
